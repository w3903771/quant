## 01 量化投资全景与简史
- 1.1 什么是量化投资：与主观投资的边界、Alpha 与 Beta、买方/卖方
- 1.2 简史脉络：规则与技术分析 → 统计套利（Thorp、文艺复兴 Medallion） → 因子投资（Fama-French、Barra） → 机器学习 → 深度学习 → 大模型/Agent
- 1.3 信号到收益的链路：预测 → 组合构建 → 交易执行 → 风控 → 绩效归因
- 1.4 信噪比、非平稳、对抗性：量化为什么"难"
- 1.5 **【独立重点节】A股制度与微观特异性**（相对英文教材的核心差异化价值，评审 P0）：T+1、涨跌停（主板±10%/创业板科创板±20%/ST±5%）、停复牌、ST/退市、融资融券与融券稀缺、北向资金（沪深股通）、龙虎榜、注册制冲击、壳价值与小市值效应的历史变迁、量化新规与监管口径——逐项说明对策略/因子/回测的具体影响
- 1.6 中美市场对比导览（制度、数据、参与者结构、有效性差异）

## 02 数据与特征工程
- 2.0 **市场微观结构基础**（评审 P0）：订单簿(LOB)、买卖价差、市场冲击、流动性、做市与撮合机制、tick/分钟/日频的数据粒度——为执行成本(08/11)与高频另类特征打底
- 2.1 数据类型：行情(OHLCV)/基本面/财务/宏观/**另类数据细分**（新闻、研报、舆情、卫星、产业链、**资金流、龙虎榜、北向资金、分析师一致预期、股东户数、互动易问答**）
- 2.2 A股数据生态：akshare / tushare / baostock / JointQuant / Qlib 数据格式；复权、停牌、ST、涨跌停、除权除息
- 2.3 三大偏差：未来函数(look-ahead)、幸存者偏差(survivorship)、point-in-time 与财报披露滞后
- 2.4 特征工程：技术指标、量价因子、横截面标准化/中性化（行业、市值）、去极值、缺失处理
- 2.5 标签构造：未来收益、超额收益、分类 vs 回归、三重门(triple-barrier)与元标签(meta-labeling, Lopez de Prado)
- 2.6 数据泄漏的隐蔽形式（特征/标签/CV 三层）

## 03 经典金融工程基石
- 3.1 有效市场假说(EMH)与异象(anomalies)
- 3.2 CAPM、APT、随机折现因子(SDF)视角
- 3.3 均值-方差与 Markowitz 前沿；Sharpe/Sortino/Calmar
- 3.4 组合优化：Markowitz、最大分散、风险平价、Black-Litterman、约束与稳健优化
- 3.5 风险模型：Barra 多因子风险模型、协方差收缩(Ledoit-Wolf)
- 3.6 **规则型与统计型择时**〔QuantsPlaybook 大量复现〕：RSRS/QRS(阻力支撑相对强度+分位回归)、低延迟趋势线、扩散指标、均线交叉/通道突破、鳄鱼线、价量共振、牛熊线、情绪/形态择时——A股研报最常见的一脉，给原理与适用边界
- 3.7 这些"老地基"在 ML 时代为何仍是底座

## 04 因子投资与 Alpha 挖掘
- 4.1 因子动物园：价值/动量/质量/低波/规模/盈利；Jegadeesh-Titman 动量、Carhart 四因子、Fama-French 五因子
- 4.1b **行为金融与微观结构因子**〔QuantsPlaybook 大量复现〕：凸显理论 STR、聪明钱、APM(隔夜/日间分离)、振幅隐藏结构、处置效应 CGO、筹码分布 CYQ、特质波动率、高频价量相关；**网络类因子**（中心度、lead-lag 滞后网络）——行为/微观成因 + 与经典因子的区别
- 4.1c IPCA（工具化主成分/条件因子模型，Kelly-Pruitt-Su）：让因子载荷随个股特征变化
- 4.2 因子检验：IC/RankIC、IR、ICIR、分层回测、换手与容量、t值与多重检验
- 4.3 因子合成：等权/IC加权/**半衰期IC加权**/最大化ICIR(协方差/Ledoit收缩)/回归法；正交化与共线性〔QuantsPlaybook composition_factor〕
- 4.4 公式化 Alpha：WorldQuant Alpha101、GTJA191、遗传规划自动挖因子
- 4.5 因子拥挤、衰减与失效；A股因子的特异性（壳价值、政策、小市值效应变迁）
- 4.6 从"线性因子"到"非线性预测"的过渡——引出机器学习

## 05 传统机器学习方法
- 5.1 把选股变成监督学习：横截面 vs 时序、回归 vs 排序(learning-to-rank)
- 5.2 线性与正则：OLS/Ridge/Lasso/ElasticNet、分位数回归
- 5.3 树模型与梯度提升：随机森林、GBDT、XGBoost、LightGBM、CatBoost（量化主力）
- 5.4 SVM/核方法、贝叶斯、kNN 等
- 5.5 无监督：PCA/因子降维、聚类(板块/风格)、异常检测
- 5.6 时序交叉验证的正确姿势：Purged K-Fold、Combinatorial Purged CV(CPCV)、embargo
- 5.6b **元标签与交易级信号过滤**（López de Prado meta-labeling；abu UMP 主裁/边裁范式）：在"是否下单/下多大"上再训一层 ML 拦截器，把方向预测与执行决策解耦
- 5.7 标杆研究：Gu-Kelly-Xiu(2020) "Empirical Asset Pricing via ML"；ML 在 A股的实证
- 5.8 **因果推断在量化**（评审 P1）：相关≠因果、双重机器学习(DML)、工具变量、事件研究法；用于策略稳健性与"伪因子"识别
- 5.9 传统 ML 的天花板：特征工程依赖、时序结构难捕捉
